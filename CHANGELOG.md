# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-02

### Added

- Initial release of FitForge
- AI-powered workout generation using Google Gemini
- Conversational AI fitness coach with LangChain
- User authentication with biometric support
- 5-step onboarding flow
- Live workout tracking with timers
- Exercise library with 50+ exercises
- Progress tracking and statistics
- Dark theme UI with smooth animations
- Offline mode support

### Backend

- Express API with TypeScript
- Supabase integration for database and auth
- PostgreSQL database with Row Level Security
- RESTful API endpoints
- LangChain integration for AI features

### Mobile

- React Native 0.81 with Expo SDK 54
- TypeScript throughout
- React Navigation for routing
- React Native Reanimated for animations
- Biometric authentication support
- AsyncStorage for offline caching

## [Unreleased]

- Nutrition tracking

- Video exercise demonstrations
